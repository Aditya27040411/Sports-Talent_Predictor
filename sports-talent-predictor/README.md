## 📌 Problem Statement

Most children aged 5–10 begin sports through trial-and-error, parental preference, or school availability — not by matching their natural abilities to sport demands. This leads to:

- Early dropout rates due to poor fit
- Missed talent at elite levels
- Frustrated children and parents

**SportTalent AI** addresses this by using scientifically grounded physical and cognitive assessments to recommend the top sports each child is most naturally suited for — before they waste years in the wrong discipline.

---

## 🧪 How It Works

```
Child undergoes 20-min assessment → Scores fed into ML model → Top-3 sport recommendations generated
```

### Input Features (7 attributes, scored 1–10)

| Feature | Description | Test Used |
|---|---|---|
| **Speed** | Explosive and sustained movement speed | 20m sprint |
| **Strength** | Muscular force output | Standing broad jump / push-up test |
| **Endurance** | Aerobic/cardiovascular capacity | 6-min run / Cooper test |
| **Flexibility** | Range of motion | Sit-and-reach |
| **Cognitive Ability** | Decision-making, spatial awareness | Reaction-time + Stroop test |
| **Reflex** | Involuntary response speed | Ruler drop test |
| **(Planned) Muscle Fiber Type** | Fast-twitch vs slow-twitch ratio | Fine-needle microbiopsy (clinical) |

### Output — Sports Predicted (7 classes)

`Basketball` · `Football` · `Tennis` · `Gymnastics` · `Long Distance Running` · `Wrestling` · `Chess`

## 🚀 Quick Start

```bash
# 1. Clone and install
git clone https://github.com/YOUR_USERNAME/sports-talent-predictor.git
cd sports-talent-predictor
pip install -r requirements.txt

# 2. Train all models
python src/train.py

# 3. Predict for a single child
python src/predict.py \
  --speed 7.5 --strength 6.0 --endurance 8.2 \
  --flexibility 5.5 --cognitive 7.0 --reflex 6.8

# Output:
# 🏅 Top 3 Sport Recommendations:
#   1. Long Distance Running  (confidence: 82%)
#   2. Football               (confidence: 11%)
#   3. Basketball             (confidence:  5%)
```

## 🧬 Dataset

- **Source**: Synthetically generated by the author, grounded in research literature on sport-specific physical demands
- **Size**: 542 records, 7 sports, 6 features
- **Generation method**: Reference sport profiles constructed from peer-reviewed sports science papers → Gaussian noise applied per attribute per sport → GAN augmentation for diversity
- **Planned expansion**: Real-world pilot data from school assessments (ethics approved)

## 🔬 Methodology

```
1. Feature Engineering   →  Normalize 1–10 scores, handle missing values
2. Data Augmentation     →  CTGAN for minority class balancing
3. Model Training        →  RF + SVM + Stacked Ensemble with CV
4. Explainability        →  SHAP values per prediction
5. Evaluation            →  F1, Kappa, confusion matrix, pilot agreement
```

## ⚠️ Limitations & Ethics

- Dataset is synthetic; real-world validation is in progress
- Model should be used as guidance only — not a deterministic verdict
- Physical attributes at 5–10 are not fixed; reassessment at 12–14 recommended
- Does not currently account for access barriers (economic, geographic)
